Overview

This app helps English learners memorize vocabulary words and short phrases, including spelling, English meaning, Chinese meaning, IPA American pronunciation marks when available, and audio pronunciation. It can instantly translate an English word or short phrase to Chinese meaning, can use Chinese input to find likely English terms, and can save user-specific vocabulary and quiz progress. Each requirement is marked with a unique ID so it can be tracked clearly.

Phase 0 validation note, 2026-05-24: iPhone is the primary use case and Windows is the automation/stress-test fallback. Android remains in scope long term but is deferred until the end. The Windows PWA dictionary POC passed with the current generated SQLite dictionary, and the real iPhone 17 Pro PWA POC was reported to work well with fast web-app startup and fast dictionary loading while online. Automated iPhone reports later verified service worker readiness, IndexedDB dictionary persistence, OPFS persistence, encrypted export/import, mock sync, and lookup p95 far under 1 second after the dictionary is open. A later iPhone offline test found that the app shell opens without Wi-Fi, but the original POC could not load/search the dictionary offline until the dictionary was persisted locally. The POC now saves the dictionary to IndexedDB after online load and falls back to that copy offline; this passed on Windows fallback automation and the iPhone suite confirms persistence. The current `sql.js` POC is not sufficient for the production iPhone DRAM target because it loads the full dictionary into JS/WASM memory. Remaining Phase 0 validation should focus on true Wi-Fi-off Home Screen dictionary search confirmation, real Google Drive OAuth sync, measured iPhone memory behavior, and a production dictionary engine such as `wa-sqlite`+OPFS or a sharded dictionary package.
